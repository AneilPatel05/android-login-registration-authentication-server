
# RESTful API using Node.js and MongoDB

To Run Project
--------
1.Switch to __node-login__ directory. Then,
```
cd node-login
```
2.Download dependencies
```
npm install
```
3.Run the project
```
node app
```
Now visit http://127.0.0.1:8080/api/v1 in your browser.
